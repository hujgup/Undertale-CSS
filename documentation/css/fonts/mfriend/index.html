Monster Friend

Monster Friend is the font "used" in the title for Undertale. Since the title is a pre-rendered image, this font cas created based on the style of the image.

It looks like this in its basic foreground form.

It looks like this in its basic background form.

It looks like this in its complex form.

Certain letters, when capitalized, have hearts cut out of their middle, like this:

mOnster frienD

For details about how to format complex titles in your HTML, please refer to the class documentation for ut-title.

The following glyphs have a heart in their center when capitalized:

Glyph Lowercase Uppercase
A a A
B b B
D d D
O o O
P p P
Q q Q
R r R

To use this font in "font-family" declarations, use "Monster Friend Fore" for the foreground version, or "Monster Friend Back" for the background version.

Reccomended fallback fonts are as follows:

  • Monster Friend Fore (for Monster Friend Back)
  • Monster Friend Back (for Monster Friend Fore)
  • Arial
  • sans-serif

The following classes make use of this font as their default:

  • ut-title
  • ut-title-basic-fore (Fore only)
  • ut-title-basic-back (Back only)
