MNC and Crypt of Tomorrow are the fonts used for the name of the player in battles.

To use this font in "font-family" declarations, use "MNC" or "Crypt of Tomorrow", respectively.

Reccomended fallback fonts are as follows:

  • Crypt of Tomorrow (for MNC)
  • MNC (for Crypt of Tomorrow)
  • Broadway
  • Arial Black
  • Arial
  • sans-serif

The following classes make use of this font as their default:

  • ut-stats (MNC)
  • ut-stats-full (Crypt of Tomorrow)
