dOcumentation - font

Mercy

Mercy is the font "used" in the action buttons during a fight (FIGHT, ACT, ITEM, and MERCY). Since these buttons in-game were custom-made sprites, this font was created based on the style of the sprites.

It looks like this. Capital letters have more space after them but are otherwise identical.

The following glyphs have non-standard rendering under this font:

Glyph Render
@ @
# #
$ $
% %
& &

To use this font in "font-family" declarations, use "Mercy".

Reccomended fallback fonts are as follows:

  • Determination Sans
  • 8-bit Operator
  • FixedSys
  • monospace (for Determination Mono)
  • sans-serif (for Determination Sans)

The following classes make use of this font as their default:

  • ut-action
