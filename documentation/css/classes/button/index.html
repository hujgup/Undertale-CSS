dOcumentation - class

ut-button

ut-button represents the wrapper around the action buttons during fights (FIGHT, ACT, ITEM, MERCY).

It looks like this.

It should typically be used in conjunction with ut-action and ut-link.

The CSS is as follows:

.ut-button {
  display:inline-block;
  text-transform:uppercase;
  text-align:right;
  border:0.1em solid;
  #f68228;
  text-decoration:none;
  padding:0.1em;
  margin-top:initial;
  margin-bottom:0.1em;
  font-size:1.5em;
}
.ut-button:link, .ut-button:visited, .ut-button:hover, .ut-button:active {
  text-decoration:none;
}
.ut-button:hover, .ut-button:active {
  border-color:#fff434;
}
