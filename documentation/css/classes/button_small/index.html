ut-button-small

ut-button-small represents the same thing that ut-button does, except it is more compact than the game-accurate ut-button.

It looks like this.

It should typically be used in conjunction with ut-action and ut-link.

The CSS is as follows:

.ut-button-small {
  display:inline-block;
  text-transform:uppercase;
  text-align:right;
  border:0.1em solid;
  #f68228;
  text-decoration:none;
  padding:0.1em;
  margin-top:initial;
  margin-bottom:0.1em;
  font-size:1em;
}
.ut-button-small:link, .ut-button-small:visited, .ut-button-small:hover, .ut-button-small:active {
  text-decoration:none;
}
.ut-button-small:hover, .ut-button-small:active {
  border-color:#fff434;
}
