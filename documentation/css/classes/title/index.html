ut-title

ut-title represents titles that should look like the Undertale logo.

It looks like this.

ut-title relies on additional HTML code to function fully.

ALERT: All of the following features will break unless you enclose your title element inside a block-level element where either its class is ut-title-parent, or it has the CSS property "position:relative;" set.

To overlay white text on the element, add the attribute "data-text" to your title element and copy the element's text content into it.

Like so

To fill in the hearts behind certain letters, add the attribute "data-hearts" to your title element, copy the element's text content into it, and at the point where you want a heart fill, use a different glyph. Cut the text short after the last heart in the title.

Like sO

Letters that may have hearts in them are listed in the Monster Friend font documentation.

The CSS is as follows:

.ut-title {
  color:#888;
  font-family:"Monster Friend Back","Monster Friend Fore",Arial,sans-serif;
  font-size:2.35em;
}
.ut-title::before, .ut-title::after {
  font-family:"Monster Friend Fore","Monster Friend Back",Arial,sans-serif;
  position:absolute;
  left:0;
  top:0;
}
.ut-title::before {
  color:#e90111;
  content:attr(data-hearts);
}
.ut-title::after {
  color:white;
  content:attr(data-text);
}
