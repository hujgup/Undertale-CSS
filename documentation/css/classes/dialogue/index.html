dOcumentation - class

ut-dialogue

ut-dialogue represents dialogue said by characters who do not have their own specific font outside of fights, as well as the object inspection text.

It looks like this.

The CSS is as follows:

.ut-dialogue {
  color:white;
  font-family:"Determination Mono","Determination Sans","8-bit Operator","FixedSys","Terminal Vector","Courier New",monospace;
  font-size:1em;
}
