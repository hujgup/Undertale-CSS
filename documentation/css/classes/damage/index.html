dOcumentation - class

ut-damage

ut-damage represents the text shown when a monster takes damage.

It looks like this (123456789).

It should typically be used in conjunction with ut-chara and ut-heading (123456789).

The CSS is as follows:

.ut-damage {
  color:white;
  font-family:Hachicro,"OCR A Extended","OCR A","Goudy Stout",Impact,Arial,sans-serif;
}
