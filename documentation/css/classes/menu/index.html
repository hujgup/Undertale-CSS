ut-menu

ut-menu represents text in menu screens, such as the [C] menu, player inventory/boxes, and sub-option selectors in battles.

It looks like this (unchanged from most of the font on this page because it is also using ut-menu).

The CSS is as follows:

.ut-menu {
  color:white;
  font-family:"Determination Mono","Determination Sans","8-bit Operator","FixedSys","Terminal Vector","Courier New",monospace;
  font-size:1em;
}
